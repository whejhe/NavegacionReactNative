Instalar Expo de manera Global desde cmd:
npm install -g expo-cli

Crear Proyecto:
npx create-expo-app@latest RegistroFirebaseExpo --template blank
npm install react-native expo

Habilitar para web:
npx expo install react-dom react-native-web @expo/metro-runtime

Navegacion:
npm install @react-navigation/native @react-navigation/stack @react-navigation/bottom-tabs @react-navigation/drawer

Animaciones y Transiciones:
npm install react-native-reanimated react-native-gesture-handler react-native-screens react-native-safe-area-context @react-native-community/masked-view

Authenticacion:
npm install firebase react-auth-code-input react-native-keyboard-avoiding-view

Dependencias de Estado:
npm install redux react-redux redux-thunk

Almacenamiento de Datos:
npm install @react-native-community/async-storage realm

Gestion de Red:
npm install axios react-native-url-polyfill

Gestion de Estilos:
npm install styled-components react-native-vector-icons

Comenzar Proyecto:
npm start

Para estilos y formato:
npx expo lint

npx expo install -- --save-dev prettier eslint-config-prettier eslint-plugin-prettier
